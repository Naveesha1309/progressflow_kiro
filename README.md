# ProgressFlow - Monthly Progress Tracker

A minimalistic web application for tracking and sharing your daily progress with a beautiful calendar interface.

## Features

### Core Functionality
- **Daily Progress Logging**: Log your accomplishments with work/personal tags
- **Calendar View**: Elegant year-long calendar showing all your progress entries
- **Privacy Controls**: Make entries public or private
- **AI-Powered Insights**: Automatic skill detection and progress visualization
- **Social Network**: Follow others and share your journey
- **Streak Tracking**: Monitor your consistency with streak counters

### Pages

1. **Home**: Quick access to log today's progress with an inspiring interface
2. **Progress**: Full-year calendar view with color-coded entries
3. **Summary**: Monthly statistics, charts, and AI-detected skills
4. **Network**: Manage followers, following, and connection requests
5. **Notifications**: Stay updated with activity and milestones
6. **Profile**: Personal information, stats, and social links

## Getting Started

1. Open `index.html` in your web browser
2. Click "Log Today's Progress" to create your first entry
3. Select work or personal category
4. Write what you accomplished (max 100 words)
5. Choose public or private visibility
6. View your progress in the calendar

## Data Storage

Currently uses browser localStorage for data persistence. In production, this would connect to a backend database with user authentication.
